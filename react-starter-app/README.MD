# React Starter App
This is based off a tutorial on creating your own react starter app. 
https://www.javascriptstuff.com/build-your-own-starter

## Usage
---
 
Start the development server with this command:
 
```
npm start
```
 

## Setup
---
 
```
npm install
```
 
 
 
## Compile
---
 
```
npm run compile
```
